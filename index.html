▸ How it works

BFS — Breadth-First Search
BFS explores nodes level by level, using a FIFO queue. It visits all neighbours at the current depth before moving deeper. This guarantees the path with the fewest stops. Edge costs are ignored — only hop count matters.

Queue: [start] → dequeue → enqueue all unvisited neighbours → repeat.

⚡ Best for: Urgent response, minimizing room transitions.
